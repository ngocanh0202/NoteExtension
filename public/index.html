Setting Environment Variables

How to get Firebase config:
  1. Go to Firebase Console
  2. Select your project (or create a new one)
  3. Click ⚙️ Settings → Project settings
  4. Scroll to Your apps → Select Web app
  5. Copy values from the firebaseConfig object

How to get Cloudinary config:
  1. Sign up at Cloudinary
  2. Dashboard: copy Cloud Name, API Key, API Secret
  3. Go to Settings → Upload tab
  4. Scroll to Upload presets → Add upload preset
  5. Set Signing Mode to Unsigned
  6. Save and copy the Preset Name
